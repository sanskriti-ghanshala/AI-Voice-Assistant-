# 🎙️ Ana - AI Voice Assistant

Ana is a Python-based AI Voice Assistant that performs voice-controlled tasks such as opening websites, playing music, telling the current time, and answering user queries using Google's Gemini AI. It uses speech recognition for voice input and Google Text-to-Speech (gTTS) for natural voice output.

---

## ✨ Features

- 🎤 Voice Command Recognition
- 🤖 AI-powered responses using Gemini API
- 🔊 Natural voice output with gTTS
- 🌐 Open popular websites
  - Google
  - YouTube
  - Facebook
  - Instagram
  - WhatsApp
  - Stack Overflow
  - Myntra
  - Flipkart
  - Chrome
  - Firefox
  - Edge
- 🎵 Play random music from your local Music folder
- ⏹️ Stop music with voice command
- 🕒 Tell the current time
- 📧 Email sending support (SMTP)
- 👋 Exit the assistant using voice commands like:
  - Stop
  - Exit
  - Bye

## 📦 Required Libraries

Install the required packages using:

```bash
pip install SpeechRecognition
pip install google-generativeai
pip install gtts
pip install pygame
pip install wikipedia
pip install pyaudio
```

Or install all at once:

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Voice-Assistant.git
```

2. Open the project folder

```bash
cd AI-Voice-Assistant
```

3. Add your Gemini API Key.

4. Run the program

```bash
python main.py
```
